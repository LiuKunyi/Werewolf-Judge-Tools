狼人杀法官助手

一个为线下狼人杀主持人设计的本地法官工具。

无需安装、无需联网，打开 HTML 文件即可使用。支持手机和电脑，可用于身份分发、夜间记录、发言计时、投票统计以及胜负判定等流程，减少法官手工记录和计算的负担。

使用方法

1. 配置本局身份

在首页选择本局需要使用的角色及人数，并设置狼人胜利规则（屠边或屠城）。

配置完成后点击“开始游戏”。

2. 分发身份

法官依次将设备交给每位玩家查看身份。

玩家确认身份后点击“知道了”，继续传递给下一位玩家。

所有身份分发完成后进入身份表。

3. 夜晚阶段

身份表会显示当前所有玩家身份。

法官可以：

* 标记夜间行动目标
* 标记出局玩家
* 查看阵营状态
* 管理特殊角色标记

确认夜间操作完成后点击“天亮”。

4. 白天阶段

系统会自动显示昨晚结算结果。

随后进入发言环节，内置发言计时器，可随时重新计时。

发言结束后点击“结束发言环节”进入投票。

5. 投票阶段

法官按顺序记录玩家投票。

支持：

* 正常投票
* 弃票
* 平票
* PK投票

投票结束后自动生成票流并统计结果。

6. 胜负判定

系统会根据场上存活情况自动判断胜负。

当满足胜利条件时，将自动显示胜利阵营。

适用场景

* 线下狼人杀聚会
* 桌游店主持
* 校园狼人杀活动
* 朋友聚会娱乐局
* 需要法官主持的标准狼人杀对局

特殊角色说明

本工具不会代替角色技能结算，仅用于记录和辅助管理部分特殊角色状态。

丘比特

首夜可点击身份表中的阵营标记，对玩家阵营进行调整。

当场上存在丘比特时，其他玩家也可以被标记为第三方阵营。

带有金色编号标记的玩家可用于记录情侣身份。

⸻

混血儿

首夜可点击其阵营标记切换所属阵营。

⸻

野孩子

可随时点击其阵营标记切换所属阵营。

带有金色编号标记的玩家可用于记录其榜样身份。

⸻

咒狐

当好人或狼人达成胜利条件时，若咒狐仍存活，则判定为第三方胜利。

⸻

夜间行动标记

身份表支持夜间行动记录。

根据本局存在的角色，会自动出现对应标记选项，例如：

* 🐺 狼刀目标
* 🛡️ 守护目标
* 🧪 女巫目标
* 💤 摄梦目标
* 🪶 乌鸦目标
* 🌹 狼美人目标
* 😈 梦魇目标
* 🎁 奇迹商人目标
* 🎭 魔术师目标
* ⭐ 自定义特殊标记

这些标记仅用于法官记录，不会自动执行角色技能效果。

⸻

永久标记

当场上存在丘比特或野孩子时，可点击玩家编号添加金色永久标记。

用于记录：

* 情侣
* 榜样
* 其他需要整局保留的信息

永久标记不会因昼夜切换而消
