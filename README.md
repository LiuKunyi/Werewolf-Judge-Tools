# 🐺 狼人杀法官工具

一个专为线下面杀设计的纯前端狼人杀法官工具。

无需安装、无需联网，打开PWA应用即可使用。

**在使用之前，请务必仔细阅读本文件，以免操作不当。**

适用于普通狼人杀、进阶板子及自定义板子，帮助法官完成身份分发、夜晚记录、阵营管理、投票统计和胜负判定等工作。

# 功能简介

## 🎭 身份配置

- 自定义本局所有身份
- 支持快速添加和删除身份
- 内置多种预设配置
- 自动统计总人数

## 🃏 身份分发

- 随机分配身份
- 逐位查看身份
- 身份颜色区分阵营
  - 好人：绿色
  - 狼人：红色
  - 第三方：灰色
- 支持提前查看身份表

## 🌙 夜晚管理

### 自动统计

- 第几晚
- 存活好人数
- 存活狼人数
- 存活第三方人数

### 夜晚行动顺序提示

根据本局实际身份动态生成行动顺序。

### 出局标记

点击身份表中玩家前方的圆圈即可标记出局，点击已出局的玩家则移除出局标记

### 夜间标记

支持：

- 🐺 狼刀目标
- 🛡️ 守护目标
- 🧪 女巫目标
- 💤 摄梦目标
- 🌹 狼美人目标
- 😈 梦魇目标
- 🎁 商人目标
- 🎭 魔术师目标
- 🔇 禁言目标
- ⚔️ 猎魔目标

特点：

-同类标记全场唯一
-魔术师允许同时存在两个目标
-已出局角色自动隐藏对应技能
-不允许的目标自动隐藏对应技能

### 永久标记

仅丘比特局或野孩子局可用。

点击需要标记玩家的序号可添加永久标记，再次点击可取消。

可用于标记：

- 情侣
- 榜样

## ⚖️ 阵营管理

当以下角色存在于本局时，支持切换阵营（点击身份表中的阵营标记）：

- 丘比特
- 混血儿
- 野孩子

方便处理特殊板子及变阵营情况。

## ☀️ 白天流程

支持：

- 夜晚结算
- 发言计时器
- 玩家投票
- 弃票
- PK发言
- PK 投票
- 票流统计

自动生成投票结果页面。

### ⏱️ 发言计时器

功能：

-0.1秒精度显示
-点击重新开始计时
-自动计时

### 📊 投票系统

支持：

-普通投票
-PK投票
-弃票
-自动统计票数
-自动生成票流
-平票自动进入 PK 流程

## 🏆 胜负判定

### 好人胜利

- 所有狼人阵营角色出局

### 狼人胜利

支持：

- 狼控场自动胜利
- 屠边和屠城模式切换

### 狼控场规则

当：

- 狼人人数 ≥ 好人人数

且场上不存在：

- 女巫
- 骑士
- 摄梦人
- 猎魔人

时，自动判定狼人胜利。

### 第三方胜利

其他阵营全部出局（咒狐、恶魔除外）。

#### 咒狐

只要咒狐或恶魔存活至游戏结束，无论最终由哪一方达成胜利条件，均直接判定第三方胜利。

# 使用流程

## 1. 配置身份

选择本局使用的身份和人数。

## 2. 开始游戏

点击 **开始游戏**。

依次让玩家查看自己的身份。

（如法官想自己告诉大家身份，可在查看第一个玩家身份后，在法官点击处理页面点击下方 **查看身份表** 按钮。）

## 3. 查看身份表

全部身份分发完成后进入身份表页面。

法官可在此记录夜间信息。

## 4. 夜晚记录

使用标记系统记录：

- 狼刀
- 女巫
- 守卫
- 摄梦人
- 狼美人
- 梦魇
- 奇迹商人
- 魔术师
- 禁言长老
- 猎魔人

的角色行动。

**点击玩家前方圆圈可标记死亡，再次点击可取消。**

## 5. 天亮结算

### 自动技能结算

💤 摄梦人

- 连续两晚摄梦同一目标，目标死亡
- 若摄梦人夜间死亡，当晚梦游目标死亡

🌹 狼美人

记录魅惑目标。
- 狼美人死亡时，魅惑目标自动殉情

适用于：夜晚死亡、投票出局、法官手动标记死亡

⚔️ 猎魔人

从第二晚开始可发动技能。

- 如果目标为狼人阵营，目标死亡
- 如果目标为好人阵营，猎魔人死亡

女巫毒药对猎魔人无效。

点击 **天亮**。

系统自动按以上规则统计死亡玩家。

## 6. 发言

点击 **发言**。

内置计时器。

点击计时器可重新开始计时。

## 7. 投票

可记录每位玩家投票。

系统自动：

- 统计票数
- 判断平票
- 发起 PK
- 生成票流

###🤡 愚者

被放逐时，弹出技能确认。

发动技能后，本轮无人出局，后续无法再次被投票放逐

###🌕 血月使徒

当血月使徒作为最后一名狼人被放逐时，弹出技能确认。

发动技能后，本轮无人出局，白天立即结束，直接进入下一晚

## 8. 进入下一晚

完成投票后进入下一晚。

系统自动更新夜晚统计信息。

# 特殊角色说明

## 丘比特

- 可标记情侣（使用永久标记功能）
- 在第一晚支持阵营切换（所有人均可切换）

## 混血儿

- 第一晚可切换阵营

## 野孩子

- 支持动态切换阵营

## 咒狐和恶魔

- 视为第三方阵营
- 只要存活至游戏结束，则直接判定第三方胜利

## 警长竞选

- 法官需**手动组织**警上发言、警下投票
